# CYCLING

Make time as slow as you want by dividing it by some value, preferably a power
of two so everything is in sync with the clock. Make sure the result is an
integer. Trick: in 2's complement, double negation removes fractional part:

```
~~2.35215 → 2
```

Apply a modulo operator to choose how many steps.

```
(~~(t/4096)%16) → 16 step sequencer, at t/4096 speed.

```
Use sequencer to modulate pitch, for example:

```
(t<<2) * (~~(t/4096)%16)
```
